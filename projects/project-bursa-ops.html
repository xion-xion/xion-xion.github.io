Project Summary

A robust Python automation tool designed to streamline the extraction of financial annual reports from Bursa Malaysia. It solves the issue of manual data collection by providing an intelligent scraping pipeline that respects server limits, handles network instability, and automatically organizes data for analysis.

Intelligent File Renaming

Raw downloads from the web often have messy, non-descriptive filenames. I implemented a parsing logic that reads the metadata and standardizes every file to COMPANY_TYPE_YEAR.pdf (e.g., MRDIY_AR_2023.pdf). This ensures that files are instantly searchable and sortable.

State Management & Caching

To prevent IP bans and reduce bandwidth, the system maintains a persistent json state file. Before requesting a file, it checks this local registry. If a report is already downloaded, it is intelligently skipped, making subsequent runs lightning fast.

Scalable Directory Architecture

The system is designed to handle multiple companies simultaneously. It automatically generates a clean directory hierarchy, creating dedicated folders for each company code (e.g., Report/GENTING, Report/MRDIY). This allows users to batch process hundreds of reports without clutter.
