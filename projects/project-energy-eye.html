Energy Eye: Commercial Product Development

Project & Deployment Manager | Timeline: 2024 - Present

Project Management: Vendor Management Project Planning
Field Operations: Device Installation Site Assessment
UI/UX: Mobile App UI/UX User-Centric Design

Project Summary

Manage the deployment and software lifecycle of "Energy Eye." My role focuses on bridging the gap between raw hardware data and user experience, coordinating with software vendors to translate technical metrics into actionable financial insights for clients.

Hands-On Deployment & Installation

As the deployment manager, I was responsible for the end-to-end installation process. This involved site assessments, hands-on device setup, and ensuring a seamless 'zero-infrastructure' deployment for our clients, directly clipping onto existing meters without any wire cutting.

Led Mobile App UI/UX Design

I led the UI/UX design for the mobile application, focusing on translating complex energy data into actionable financial insights for business owners. The goal was to create an intuitive interface that clearly visualized cost savings, such as the 'Peak vs. Off-Peak' usage, empowering users to make informed decisions.

Software Vendor Management & Data Logic

I acted as the technical bridge between the hardware and the external software development team. I was responsible for translating raw sensor data logic into clear requirements for the developers, ensuring the app displayed accurate energy metrics. I also led the UI/UX feedback loop, gathering insights from field installations to guide the software team in building features that customers actually needed.
